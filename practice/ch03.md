### 练习3.23
  question:编写一段程序，创建一个含有10个整数的vector对象，然后使用迭代器将所有元素的值变为原先的两倍。输出vector对象的内容，检验程序是否正确。  
	answer:  
  ```cpp
	#include <iostream>
	#include <vector>

	using std::cin;
	using std::cout;
	using std::endl;
	using std::vector;

	int main(){
		vector<int> v;
		for(int i = 0; i < 10; i++)
			v.push_back(i);
		for(auto it = v.begin(); it != v.end(); ++it)
			*it *= 2;
		for(auto it = v.begin(); it != v.end(); ++it)
			cout<<*it<<" ";
		cout<<endl;
		return 0;
	}
```
