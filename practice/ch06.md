### 练习6.3 
  question:编写你自己的fact函数，上机检查是否正确。  
  answer:  
  ```cpp
#include <iostream>

using std::cin;
using std::endl;
using std::cout;

int fact(int n){
    if(n == 1) return n;
    return n * fact(n-1);
}

int main(){
    int n;
    cin>>n;
    cout<<fact(n);
    return 0;
}
```

### 练习6.4
  question:编写一个与用户交互的函数，要求用户输入一个数字，计算生成该数字的阶乘，在main函数中调用该函数。  
  answer:  
  ```cpp
#include <iostream>

using std::cin;
using std::endl;
using std::cout;

int fact(int n){
    if(n == 1) return n;
    return n * fact(n-1);
}

int main(){
    int n;
    cin>>n;
    cout<<fact(n);
    return 0;
}
```

### 练习6.5
  question:编写一个函数输出其实参的绝对值。  
  answer:  
  ```cpp
#include <iostream>

using std::cin;
using std::cout;
using std::endl;

int abs(int n){
    return n > 0 ? n : -n;
}

int main(){
    int n ;
    cin>>n;
    cout<<abs(n)<<endl;
    return 0;
}
```
