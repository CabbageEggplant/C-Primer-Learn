### 练习7.5
  question:在你的Person类中提供一些操作使其能够返回姓名和住址。这些函数是否应该是const的呢？请说明原因。   
  answer:  
  Person.h
  ```cpp
  #ifndef Person_H
#define Person_H

#include <string>

using std::string;

class Person{
private:
    string name;
    string address;
public:
    Person();
    string getName()const;
    string getAddress()const;
};


#endif
```
  Person.cpp
  ```cpp
  #include "Person.h"

Person::Person(){

}

string Person::getName()const{
    return name;
}

string Person::getAddress()const{
    return address;
}

int main(){
    return 0;
}
```
